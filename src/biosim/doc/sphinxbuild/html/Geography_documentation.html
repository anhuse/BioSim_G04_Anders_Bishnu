CreateMap¶

CreateMap class

class geography.CreateMap(geo_matrix_input_string)[source]¶

Class for the islands geography. This class:

  • Stores the size/shape of the geography of the island.
  • Converts string input of map, consisting of letters corresponding to landscape types into a 2D matrix of objects.
  • Objects can only be one of the five Landscape types: Ocean (O), Savannah (S), desert (D), jungle (J) and mountain (M).
Variables:
  • geo_matrix_input_string – str, String with map coordinates
  • geo_graph – str, input string without whitespace
  • lines – list, list of each line of the input string
  • geo_shape – tuple(default, None), Shape of the geography
  • object_matrix – list, 2D list of cell objects
  • geo_list – list, 2D list of string characters
  • first_row – str, characters of the first row of the map
  • last_row – str, characters of the last row of the map
  • first_column – list, characters of the first column of the map
  • last_column – list, characters of the last column of the map

When instantiated, this class:

  • Checks if input characters are valid letters
  • Checks if all rows in map have equal length
  • Checks that ocean (“O”) is around all edges of the map
Parameters:geo_matrix_input_string – str, String with map coordinates
