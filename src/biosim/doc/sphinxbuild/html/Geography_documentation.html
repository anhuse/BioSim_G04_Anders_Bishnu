The Geography Class and its Subclasses¶

Geo class

class Geography.Geo(geo_matrix_input_string)[source]¶
  • Stores the size/shape of the geography of the island
  • Converts string input of map, consisting of letters corresponding to

landscape types into a 2D matrix of objects - Objects can only be one of the five Landscape types: Ocean (O), Savannah (S), desert (D), jungle (J) and mountain (M)

  • Checks if input characters are valid letters
  • Checks if all rows in map have equal length
  • Checks that ocean (“O”) is around all edges of map
Parameters:geo_matrix_input_string – String with map coordinates
